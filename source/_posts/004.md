### 使用 Git 命令行（推荐）

前提：你的电脑上需要安装好[Git](https://git-scm.com/)。

#### 第一步：在 GitHub 上创建新仓库

1. 登录 GitHub，点击右上角的 **+** 号，选择 **New repository**。
2. 输入仓库名称。
3. **不要**勾选 "Initialize this repository with a README/gitignore/license"（为了避免后续的冲突，我们创建一个空仓库）。
4. 点击**Create repository**。
5. 复制生成的仓库 HTTPS 地址（例如`https://github.com/你的用户名/仓库名.git`）。

#### 第二步：在本地初始化 Git

1. 打开你的项目文件夹。
2. 右键选择**Git Bash Here**（或者在终端中`cd` 进入该目录）。
3. 输入以下命令初始化仓库：

```C++
git init
```

#### 第三步：添加文件并提交

回到命令行，依次执行：

1. **添加所有文件**：

```C++
git add .
```
2. **提交到本地仓库**：

```Bash
git commit -m "第一次提交"
```

#### 第四步：关联远程仓库并推送

将第一步里复制的 URL 替换掉下面的地址：

1. **切换分支名**（现在的 GitHub 默认主分支叫 main）：

```Bash
git branch -M main
```
2. **关联远程仓库**：


```Bash
git remote add origin https://github.com/你的用户名/仓库名.git
```
3. **推送到 GitHub**：


```Bash
git push -u origin main
```

*注意：如果你是第一次使用，它会弹窗让你登录 GitHub 账号。*




### 后续如果想添加文件，只需 3 步

只要你的本地文件夹已经是 Git 仓库（之前做过 `git init`），后续每次添加或修改文件，只需要重复执行下面这三行命令：

#### 1. 添加（存入暂存区）

将所有变动（新文件、修改过的文件）放入“待提交清单”。

```Bash
git add .

```

> **注意：** `.` 代表当前目录下的所有变动。如果你只想添加特定文件，把 `.` 换成文件名。

#### 2. 提交（保存到本地仓库）

将暂存区的内容正式保存，并附上一句说明。

```Bash
git commit -m "这里写你的修改备注，比如：添加了登录功能"

```

#### 3. 推送（同步到 GitHub）

将本地的新提交发送到远程 GitHub 服务器。

```Bash
git push

```

> **注意：** 后续不需要再加 `-u origin main`，直接输入 `git push` 即可，Git 会记住默认的上传目标。
