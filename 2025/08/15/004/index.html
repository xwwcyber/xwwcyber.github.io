怎么把代码传到自己github仓库中

编辑原文
2025-08-15 tips 320 字
github

使用 Git 命令行（推荐）

前提：你的电脑上需要安装好Git。

第一步：在 GitHub 上创建新仓库

  1. 登录 GitHub，点击右上角的 + 号，选择 New repository。
  2. 输入仓库名称。
  3. 不要勾选 “Initialize this repository with a README/gitignore/license”（为了避免后续的冲突，我们创建一个空仓库）。
  4. 点击Create repository。
  5. 复制生成的仓库 HTTPS 地址（例如https://github.com/你的用户名/仓库名.git）。

第二步：在本地初始化 Git

  1. 打开你的项目文件夹。
  2. 右键选择Git Bash Here（或者在终端中cd 进入该目录）。
  3. 输入以下命令初始化仓库：
git init

第三步：添加文件并提交

回到命令行，依次执行：

  1. 添加所有文件（会自动忽略.gitignore 里的文件）：
git add .
  1. 提交到本地仓库：
git commit -m"Initial commit: Upload VLC Player project"

第四步：关联远程仓库并推送

将第一步里复制的 URL 替换掉下面的地址：

  1. 切换分支名（现在的 GitHub 默认主分支叫 main）：
git branch -M main
  1. 关联远程仓库：
git remote add origin https://github.com/你的用户名/仓库名.git
  1. 推送到 GitHub：
git push -u origin main

注意：如果你是第一次使用，它会弹窗让你登录 GitHub 账号。
