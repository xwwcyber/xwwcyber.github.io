关于markdown的基础语法

编辑原文
2025-01-28 技术文章 452 字
markdown

基础语法

1.标题

# 1级标题
## 2级标题
### 3级标题
#### 4级标题
##### 5级标题
###### 6级标题

注意#与标题中间有空格。

2.段落

这是第一段内容。

这是第二段内容（中间空了一行）。

在 Markdown 中，段落之间需要通过空行（即按两次回车）来分隔

3.换行

这是第一行。(两个空格)
这是第二行

在markdown中，换行只需要在第一行文字后添加两个空格即可。

4.强调

**加粗**
*倾斜*

在markdown中，强调有两种语法：

  1. 加粗，在需要加粗的内容前后用两个 * 进行包裹，同时注意 * 要和文字挨着。
  2. 倾斜，在需要倾斜的内容前后用一个 * 进行包裹，同时注意 * 要和文字挨着。

5.引用

> 这里是引用的内容。
> 你可以嵌套引用，或者连续使用。

在markdown中，引用需要在开头添加>然后加一个空格即可。

6.列表

有序列表

1. 内容A
2. 内容B

无序列表

- 观点A
- 观点 B
  - 观点 B 的子观点 (前面加空格缩进）

（使用 -、+ 或 * 三者中的一个加上空格都可以实现）

任务列表

- [x] 完成了需求分析
- [ ] 正在编写代码
- [ ] 等待测试

7.代码块

使用三个反引号 ``` 包裹，并指定语言（如 html, javascript, css, python）以实现代码高亮。

function hello() {
  console.log("Hello, Markdown!");
}

8.分割线

如果你想在文章中添加一条横线来分割不同的内容板块，可以使用三个或更多的短横线 **-**、星号 * 或下划线 **_**。
